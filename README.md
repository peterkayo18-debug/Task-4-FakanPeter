# DecodeLabs Data Analytics Internship — Project 4: Data Visualization (Power BI)

## 📌 Project Overview
This repository contains my submission for Project 4 (Optional Mastery Phase) 
of the DecodeLabs Data Analytics Internship (Batch 2026). The goal of this 
project was to translate findings from Projects 1–3 into a single, 
interactive Power BI dashboard capable of communicating insights clearly 
enough for executive decision-making.

---

## 🎯 Project Objective
Create visual representations of the e-commerce dataset to communicate 
insights clearly — choosing the right chart type for each business question, 
maintaining design integrity (zero-baseline axes, no 3D effects), and 
building an interactive dashboard using slicers.

---

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions) for custom measures
- Dataset: Cleaned 1,200-order e-commerce dataset (from Project 1)
